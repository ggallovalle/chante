uwu draws inspiration from several existing projects in the terminal styling space.

## Inspiration

- [styleText from `node:util`](https://nodejs.org/api/util.html#utilstyletextformat-text-options) - Node.js built-in styling
- [Bun.color](https://bun.com/docs/runtime/color) - Bun's built-in color API
- [visulima/colorize](https://visulima.com/docs/packages/colorize) - A universal color API
- [chalk](https://github.com/chalk/chalk) - The classic Node.js terminal string styling library
- [owo-colors](https://docs.rs/owo-colors/4.3.0/owo_colors/) - Rust terminal colors library

## Comparison

| Library           | Platform         | Effect Integration |
| ----------------- | ---------------- | ------------------ |
| uwu               | Node/Bun/Browser | Yes                |
| chalk             | Node.js          | No                 |
| visulima/colorize | Universal        | No                 |
| owo-colors        | Rust             | No                 |
